# スライド動画メーカー（インスタ向け）

アプリのインストール不要で、ブラウザだけで複数画像から縦長MP4動画（1080×1920px）を作れる無料ツールです。日本語・英語に対応しています。

👉 **[ツールを開く](https://nanashiai.github.io/slide-video-maker/)**

---

## こんな人向け

- InstagramリールやTikTok用のスライド動画を手軽に作りたい人
- 画像をアップロードするだけでMP4を得たい人
- アカウント登録なしで今すぐ使いたい人

---

## 主な機能

| 機能 | 説明 |
|---|---|
| MP4動画生成 | 複数画像から縦長（1080×1920px）のMP4を生成 |
| 表示時間の調整 | 1枚あたり1〜5秒で設定可能 |
| 完全ブラウザ完結 | 画像はサーバーに送信されず、端末内で処理 |
| 日英切り替え | URLパラメーター `?lang=en` で英語表示に切り替え可能 |
| 進捗表示 | エンコード中の進捗をバーで確認できる |

---

## 使い方

1. 上のリンクからツールを開く
2. 1枚あたりの表示時間を選択
3. 画像を選択（複数可）
4. 「動画を生成する」をクリック
5. 完成したら「動画を保存する」でダウンロード

## 動作環境

| ブラウザ | 対応状況 |
|---|---|
| Chrome（PC・Android） | ✅ 推奨 |
| Edge（PC） | ✅ 対応 |
| Safari（iOS・Mac） | ⚠️ 一部制限あり |
| Firefox | ⚠️ 一部制限あり |

VideoEncoder API を使用しているため、Chrome / Edge での使用を推奨します。

---

## 言語切り替え

URLに `?lang=en` を付けることで英語表示になります。

```
日本語版：ツールのURL
英語版　：ツールのURL?lang=en
```

---

## プライバシーについて

このツールはすべての処理をブラウザ内で完結します。アップロードした画像・生成した動画はサーバーに送信されません。
